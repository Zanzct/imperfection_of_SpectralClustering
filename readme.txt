验证结果分为三种：
第1种、 取第一第二特征向量大概率为此，是一种错误的分类情况。
第2种、 论文中所提到的典型错误，类间的边很多。
第3种、 理想的分类情况，取第二第三特征值向量做分类时会出现此分类结果，且为大概率出现。

结论：谱聚类确实存在一些缺陷，这种缺陷可能是相似度的不均衡导致的，但更准确的结果过还需要进行验证。
